# 📌  pom.xml 버전 관리
사용되는 모든 의존성 라이브러리들은 spring-boot 에서 제공하는 BOM 을 사용합니다.<br/>
수동적인 버전 기입은 추후에 CVE 문제를 고치는데 시간이 많이 걸릴듯하여 BOM 을 사용했습니다.<br/>
여기서 사용되는 BOM 은 spring-boot 3.3.0 에서 사용되는 것과 동일한 BOM 입니다.<br/>
